# Open Jupyter Notebook

This workflow opens Jupyter Notebook at the specified directory.

## Steps

// turbo
1. Run the following command to start Jupyter Notebook:
```bash
jupyter notebook
```
**Working directory:** `D:\CODEING`

This will open Jupyter Notebook in your default browser with `D:\CODEING` as the root directory.
